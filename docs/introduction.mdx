# Bio Programming Tools

Bio Programming Tools is a Python library providing unified interfaces to state-of-the-art bioinformatics and machine learning models for computational biology.

## Overview

This library wraps cutting-edge models and tools across multiple domains:

- **Structure Prediction**: AlphaFold3, Boltz2, Chai-1, ESMFold, Protenix, ViennaRNA
- **Inverse Folding**: ProteinMPNN, LigandMPNN
- **Sequence Scoring**: Enformer, Borzoi, AlphaGenome
- **Masked Models**: ESM2, ESM3
- **Causal Models**: Evo2, ProGen2
- **Structure Design**: RFdiffusion3
- **Structure Dynamics**: BioEMU
- **Gene Annotation**: BLAST, MMseqs2, PyHMMER
- **ORF Prediction**: Prodigal, ORFipy
- **RNA Splicing**: Splice Transformer
- **Sequence Alignment**: MAFFT, ColabFold Search

## Key Features

- **Unified Interface**: Consistent Python APIs with standardized Pydantic models
- **Isolated Environments**: Automatic virtual environment management for conflicting dependencies
- **Automatic Setup**: Model weights and dependencies downloaded on first use
- **GPU Management**: Smart device allocation for multi-GPU systems
- **Production Ready**: Comprehensive logging and error handling

## Quick Example

```python python icon="python"
from bio_programming_tools.tools.structure_prediction import run_esmfold, ESMFoldInput, ESMFoldConfig

# Predict structure
input_data = ESMFoldInput(sequences=["MVLSPADKTN"])
config = ESMFoldConfig(device="cuda")
result = run_esmfold(input_data, config)

print(f"pLDDT: {result.structures[0].plddt}")
```
