# Borzoi

## Overview
Borzoi predicts regulatory activity from long DNA context windows and supports both single-replicate and 4-replicate ensemble inference.

- Model context: `524,288` bp
- Output bins: `6,144`
- Species: `human`, `mouse`
- Tool keys:
  - `borzoi-prediction`
  - `borzoi-ensemble`

## Module Structure
Borzoi now uses one file per tool, plus a shared base module.

- `base.py`: shared constants/input model/runtime helpers
- `borzoi_prediction.py`: single-replicate data models + tool + helpers
- `borzoi_ensemble.py`: ensemble data models + tool + helpers
- `standalone/inference.py`: standalone inference entrypoint used by `EnvManager("borzoi")`
- `examples/example.ipynb`: end-to-end usage notebook (single + ensemble)
- `examples/example_output/`: example export target directory

## Python API
```python python icon="python"
from bio_programming_tools.tools.sequence_scoring.borzoi import (
    BorzoiInput,
    BorzoiConfig,
    BorzoiEnsembleConfig,
    run_borzoi,
    run_borzoi_ensemble,
)
```

## Input
| Field | Type | Description |
|---|---|---|
| `sequence` | `str` | DNA sequence, exactly `524,288` bp |

## Single-Replicate Config (`BorzoiConfig`)
| Field | Type | Default | Description |
|---|---|---|---|
| `output_tracks` | `List[int]` | required | Track indices to extract |
| `species` | `"human" \| "mouse"` | `"human"` | Species model |
| `replicate` | `"0" \| "1" \| "2" \| "3"` | `"0"` | Replicate ID |
| `avg_output_tracks` | `bool` | `True` | Average selected tracks |
| `use_flash_attn` | `bool` | `True` | Use FlashAttention variant |
| `device` | `str` | `"cuda"` | Inference device |
| `verbose` | `bool` | `False` | Verbose logging |

## Ensemble Config (`BorzoiEnsembleConfig`)
| Field | Type | Default | Description |
|---|---|---|---|
| `output_tracks` | `List[int]` | required | Track indices to extract |
| `species` | `"human" \| "mouse"` | `"human"` | Species model |
| `avg_output_tracks` | `bool` | `True` | Average selected tracks |
| `use_flash_attn` | `bool` | `True` | Use FlashAttention variant |
| `device` | `str` | `"cuda"` | Inference device |
| `verbose` | `bool` | `False` | Verbose logging |

## Outputs
### `BorzoiOutput`
| Field | Type | Description |
|---|---|---|
| `sequence` | `str` | Input sequence |
| `sequence_length` | `int` | Always `524,288` |
| `prediction` | `List[List[float]]` | Shape `[num_tracks, 6144]` |
| `output_tracks` | `List[int]` | Track indices used |
| `species` | `str` | Species used |
| `replicate` | `str` | Replicate used |
| `avg_output_tracks` | `bool` | Whether averaging was applied |

### `BorzoiEnsembleOutput`
| Field | Type | Description |
|---|---|---|
| `sequence` | `str` | Input sequence |
| `sequence_length` | `int` | Always `524,288` |
| `predictions` | `List[List[List[float]]]` | Shape `[4, num_tracks, 6144]` |
| `output_tracks` | `List[int]` | Track indices used |
| `species` | `str` | Species used |
| `avg_output_tracks` | `bool` | Whether averaging was applied |
| `num_replicates` | `int` | Always `4` |

## Quick Start
```python python icon="python"
from bio_programming_tools.tools.sequence_scoring.borzoi import (
    BorzoiInput,
    BorzoiConfig,
    run_borzoi,
)

sequence = "ATCG" * 131072  # 524,288 bp

inputs = BorzoiInput(sequence=sequence)
config = BorzoiConfig(
    output_tracks=[0, 1, 2],
    species="human",
    replicate="0",
    avg_output_tracks=True,
)

result = run_borzoi(inputs, config)
print(len(result.prediction), len(result.prediction[0]))
```

3. Export to `examples/example_output/`

## Notes
- Borzoi requires exact sequence length (`524,288`).
- `species="mouse"` requires `use_flash_attn=False`.
- Ensemble mode runs all four replicates and is slower than single-replicate mode.
