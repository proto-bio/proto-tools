# Sequence Scoring Tools

Predict functional genomic readouts directly from DNA sequence, including gene expression,
chromatin accessibility, transcription initiation, and regulatory activity. These tools
estimate how a sequence behaves and how variants change that behavior, at resolutions
ranging from individual regulatory elements to long genomic windows.

- **Input:** a DNA sequence, or a sequence variant to evaluate.
- **Output:** predicted functional and regulatory activity (such as expression or accessibility), or a variant's effect on it.
