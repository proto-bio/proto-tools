# Structure Alignment Tools

Align and compare three-dimensional protein structures to measure how similar their folds
are. Using length-independent similarity scores and superposition, these tools evaluate
predicted or designed structures, find structural homologs, and detect distant
relationships that sequence comparison alone would miss.

- **Input:** two or more 3D structures to compare, or a query structure to search against a set.
- **Output:** length-independent similarity scores and an optimal superposition quantifying how alike the folds are.
