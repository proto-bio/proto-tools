# Sequence Alignment Tools

Align sequences and search sequence databases to reveal evolutionary and functional
relationships. These tools build multiple sequence alignments that expose conserved regions
and motifs, and search large databases for homologs. The resulting alignments are also a
standard input for alignment-based structure prediction.

- **Input:** a set of related sequences, or a query sequence plus a database to search.
- **Output:** a multiple sequence alignment, or ranked homology hits with similarity statistics.
