# Gene Annotation Tools

Search and annotate sequences to discover what they are. These tools find homologs in
reference databases, label protein domains and families, and identify functional elements
such as CRISPR arrays, microRNA target sites, and promoters, turning an unannotated sequence
into a set of functional and evolutionary hypotheses.

- **Input:** a nucleotide or protein sequence.
- **Output:** annotated features and database hits, such as homologs, protein domains, and functional elements, with significance scores.
