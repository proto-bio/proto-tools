# Structure Scoring Tools

Evaluate predicted or designed protein structures with energy functions and quality
metrics. These tools assign scores that quantify structural plausibility, stability,
secondary-structure content, and the quality of interfaces in complexes, helping rank or
filter candidate structures.

- **Input:** a predicted or designed structure, either a single chain or a multi-chain complex.
- **Output:** numerical quality and energy scores covering plausibility, stability, secondary structure, and interface quality.
